# Hello World in Java

Hello World adalah sebuah program komputer yang menampilkan pesan serupa ke layar komputer saat seorang programmer mengetes hasil dari suatu bahasa pemrograman. Berikut ini kode “Hello World”

~~~java
Public class HelloWorld {
  Public static void main(String args[]) {
    System.out.println(“Hello World!”);
  }
}
~~~

Jika kita coba jalankan, kode tersebut akan menghasilkan output seperti ini:

~~~
Hello World!
~~~
